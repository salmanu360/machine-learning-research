# Research Summary

## Title
Impact of Missing Values on Machine Learning Classification Using a Mean Imputation Strategy

## Overview
This research investigates how missing values influence machine learning classification results and evaluates mean imputation as a handling strategy.

## Objective
To understand how missing data affects predictive performance and whether mean imputation can provide a practical baseline solution.

## Method
- introduce missing values into structured data
- apply mean imputation
- train classification models
- compare performance before and after imputation

## Contribution
The work highlights the practical impact of missing values and shows how imputation strategies influence machine learning outcomes.

## Keywords
missing values, mean imputation, classification, machine learning, data preprocessing
