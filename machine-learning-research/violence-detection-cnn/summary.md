# Research Summary

## Title
An Automatic Violence Detection System using Deep Convolutional Network

## Overview
This research focuses on building an automated system for detecting violent activity using deep convolutional neural networks.

## Objective
To support intelligent video analysis and improve automated identification of violence-related patterns in visual data.

## Method
- video/frame preprocessing
- visual feature learning with CNN
- classification of violent vs non-violent scenes
- evaluation of predictive performance

## Contribution
The research demonstrates the potential of deep learning approaches for surveillance and video analytics tasks.

## Keywords
violence detection, CNN, deep learning, computer vision, video analytics
