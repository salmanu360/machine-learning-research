# COVID-19 Pneumonia Classification with AdaBoost

This folder contains a simplified demonstration implementation inspired by my research on COVID-19 pneumonia classification using data augmentation and AdaBoost.

## Purpose
The goal of this demo is to show the overall machine learning workflow:
- load or simulate image-derived features
- preprocess data
- train an AdaBoost classifier
- evaluate model performance

## Important Note
This is a demonstration implementation created for portfolio and research presentation purposes. It is **not** the original experimental code used in the published paper.

## Files
- `summary.md` — research summary
- `demo-model.py` — demo classification pipeline

## Research Focus
- Medical image classification
- COVID-19 pneumonia detection
- Data augmentation
- AdaBoost
