# Research Summary

## Title
Dark Data in Accident Prediction: Using AdaBoost and Random Forest for Improved Accuracy

## Overview
This research explores the use of ensemble machine learning methods to improve accident prediction performance using structured data.

## Objective
To investigate whether AdaBoost and Random Forest can improve predictive accuracy in accident-related classification problems.

## Method
- structured data preprocessing
- model training with AdaBoost and Random Forest
- result comparison
- classification performance evaluation

## Contribution
The study highlights the usefulness of ensemble learning methods for extracting value from underutilized or incomplete accident-related data.

## Keywords
accident prediction, AdaBoost, Random Forest, machine learning, structured data
